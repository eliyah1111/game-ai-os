# Object Taxonomy

## Puzzle Objects

- `block`: square or rounded board piece
- `tile`: board tile, goal tile, hazard tile
- `ball`: circular puzzle piece or bubble-shooter object
- `tube`: color-sort bottle or tube
- `barrier`: wall, blocker, gate, frozen layer
- `booster`: bomb, rocket, hammer, shuffle, magnet
- `collectible`: star, coin, gem, key, token
- `arrow`: direction cue, motion guide, objective pointer

## Variant Rules

- Color variants should preserve silhouette and lighting.
- State variants may add overlays, locks, glows, cracks, or badges.
- Boosters may have stronger accents but still use the same shadow and radius language.
- Board pieces need the cleanest silhouettes and highest color readability.
